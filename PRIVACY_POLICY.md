# 개인정보처리방침 (Privacy Policy)

**최종 업데이트: 2026년 1월 6일**

Memex (이하 "서비스")는 사용자의 개인정보를 수집하지 않습니다.

**우리는 서버가 없기 때문입니다.**

---

## 1. 데이터의 수집 및 저장

본 서비스는 **"On-Device AI"** 기술을 기반으로 동작합니다.

- 사용자의 모든 데이터(대화 내용, 저장된 기억, 벡터 임베딩 등)는 **사용자의 컴퓨터 브라우저 내부**(IndexedDB, LocalStorage)에만 저장됩니다.
- 개발자는 이 데이터에 접근할 권한도, 방법도 없습니다.

---

## 2. 외부 데이터 전송 (Zero-Data Leakage)

본 서비스는 AI 추론을 위해 OpenAI, Anthropic 등의 **외부 클라우드 서버로 데이터를 전송하지 않습니다.**

| 기술 | 구동 방식 |
|------|----------|
| AI 모델 | Chrome 내장 Gemini Nano (**로컬 구동**) |
| 임베딩 | Transformers.js (**로컬 구동**) |
| 데이터베이스 | Orama (**로컬 구동**) |

따라서 **인터넷 연결이 끊긴 상태(오프라인)에서도 완벽하게 동작**하며, 이는 데이터가 외부로 유출되지 않음을 기술적으로 증명합니다.

---

## 3. 권한 사용 목적

| 권한 | 사용 목적 |
|------|----------|
| `activeTab` / `scripting` | 사용자가 요청한 현재 페이지의 텍스트를 읽어 요약하기 위함입니다. 이 데이터는 즉시 휘발되거나 로컬 DB에만 저장됩니다. |
| `storage` / `unlimitedStorage` | 사용자의 지식 데이터를 로컬에 영구 저장하기 위함입니다. |
| `sidePanel` | 브라우저 사이드 패널에 UI를 표시하기 위함입니다. |
| `contextMenus` | 우클릭 메뉴를 통한 빠른 AI 기능 제공을 위함입니다. |

## 5. 데이터 보안

- 모든 데이터는 사용자의 로컬 브라우저에만 저장됩니다.

- 브라우저의 보안 샌드박스 내에서 동작합니다.
